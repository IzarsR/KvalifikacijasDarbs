# Playlytic Frontend

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Development Server
```bash
npm start
```

The application will open at http://localhost:3000

## Features

- **Home Page**: Welcome page with project information
- **Videos Page**: Add and view video titles
- **Navigation**: Simple navigation between pages

## Notes

- Make sure the backend server is running on http://localhost:5000
- The application connects to the backend API to fetch and save videos
